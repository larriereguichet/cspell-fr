# cspell-fr

:fr: Une image `cspell` avec le dictionnaire Français installé par défaut.

Pour une vérification d'orthographe rapide :
```sh
docker run --rm -v $(pwd):/workdir ghcr.io/larriereguichet/cspell-fr README.FR.md
```
